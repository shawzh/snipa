# snipa
### Based on PyQT5,scapy
### A network sniff tool

only support Linux

## Installation

Execute:

```
git clone https://github.com/shawzh/snipa.git
cd snipa 
pip install requirements.txt
```

Also need:
```angular2html
sudo apt-get install texlive-latex-base
```

## Run


### must run as root!
```
sudo python mainWindow.py
```
